Azure training

Learning paths

AWS offer various learning paths on their website. We have picked the appropriate learning paths that relate to roles in Home Office. You should explore the different options in the section below and select the appropriate one depending on your experience, job role and interests. Selecting a learning path will show you all the training available for that level. You can then view the e-learning and the classroom training available.

The Cloud Practitioner learning path is intended for individuals who are looking to build and validate overall understanding of the AWS Cloud. This path is useful for individuals in technical, managerial, sales, purchasing, or financial roles who work with the AWS Cloud.

The Developer learning path is designed for software developers who want to learn how to develop cloud applications on AWS. Build technical skills as you progress along the path toward AWS Certification.

The Architect learning path is designed for solutions architects, solution design engineers, and anyone who wants to learn how to design applications and systems on AWS. Build technical skills as you progress along the path toward AWS Certification.

The Operations learning path is designed for sysops administrators, systems administrators, and those in a DevOps role who want to learn how to create automatable and repeatable deployments of applications, networks, and systems on the AWS platform. Build technical skills as you progress along the path toward AWS Certification.

The DevOps learning path is designed for individuals who want to learn how to use the most common DevOps patterns to develop, deploy, and maintain applications in the AWS Cloud. Build technical skills as you progress along the path toward AWS Certification.

How to book Azure training

The Home Office Public Cloud Team works closely with AWS to provide training. These training events are classroom based and are usually taught by AWS’s specialist trainers. All the courses will be online for the foreseeable future.

As an organisation we need to redouble our efforts in Cloud Training. Our cloud providers have engaged with us to provide free training for Home Office staff and we need to make sure we are taking full advantage of this opportunity.

There is a wide range of free AWS training available explained below. If you would like to attend one of the courses, please contact: public.cloud@homeoffice.gov.uk

Recommended courses

The Home Office Public Cloud Team works closely with AWS to provide virtual classrooom based training. We can offer free training on 9 different courses for AWS. Please find details below.

Architecting on AWS

The Architecting on AWS course is for architects who are familiar with AWS but do not have any formal certifications. It is recommended that you have first completed Cloud Practitioner level training first.

This course is recommended for:

  • Solutions Architects
  • Technical Architects
  • Anyone who needs to understand the scope of architecting on AWS
Advanced Architecting on AWS

The Advanced Architecting on AWS is intended for individuals who are experienced with designing scalable and elastic applications on the AWS platform. This course introduces specialized AWS services, including AWS Direct Connect and AWS Storage Gateway to support Hybrid architecture. It also covers designing best practices for building scalable, elastic, secure, and highly available applications on AWS.

This course is recommended for:

  • Senior/Lead Architects within Business Areas/Programmes
  • TDA Leads
Architecting on AWS Accelerator

The Architecting on AWS Accelerator course combines Architecting on AWS and Advanced Architecting on AWS to offer a comprehensive, immersive course in cloud architecture. It covers all aspects of how to architect for the cloud over 5 days. You will learn how to design cloud architectures, starting small and working to large-scale enterprise level designs—and everything in between.

This course is recommended for:

  • Senior/Lead Architects within Business Areas/Programmes
  • TDA Leads
DevOps Engineering on AWS

The DevOps Engineering on AWS teaches you how to use the combination of tools, practices, and cultural philosophy of DevOps to improve an organization’s ability to develop, deliver, and maintain applications and services at high velocity on AWS. This course covers Continuous Integration (CI), Continuous Delivery (CD), microservices, infrastructure as code, monitoring and logging, and communication and collaboration.

This course is recommended for:

  • Developers & Engineers who need to work in a cloud native environment using CI/CD
  • DevOps Engineers looking after applications and platforms hosted on cloud infrastructure
System Operations on AWS

The System Operations on AWS course is designed to teach those in a systems administrator or Development Operations (DevOps) role how to create automatable and repeatable deployments of networks and systems on the AWS platform. The course covers the specific AWS features and tools related to configuration and deployment, in addition to best practices for configuring and deploying systems.

This course is recommended for:

  • SysOps Engineers
  • DevOps Engineers looking after applications and platforms hosted on cloud infrastructure
Developing on AWS

The Developing on AWS course will teach you how to use the AWS SDK to develop secure and scalable cloud applications. It will also explore how to interact with AWS using code and learn about key concepts, best practices, and troubleshooting tips.

This course is recommended for:

  • Developers
  • Software Engineers
  • Infrastructure Engineers
Planning and Designing Databases on AWS

The Planning and Designing Databases on AWS course will teach you about the process of planning and designing both relational and nonrelational databases. By the end of this course, you will be familiar with the planning and design requirements of all 8 of these AWS databases services, their pros and cons, and how to know which AWS databases service is right for your workloads.

This course is recommended for:

  • Tech Leads
  • Lead Developers/Engineers
  • Solutions Architects
  • Technical Architects
Big Data on AWS

The Big Data on AWS course introduces you to cloud-based big data solutions such as Amazon Elastic MapReduce (EMR), Amazon Redshift, Amazon Kinesis and the rest of the AWS big data platform. In this course, we show you how to use Amazon EMR to process data using the broad ecosystem of Hadoop tools like Hive and Hue. We also teach you how to create big data environments, work with Amazon DynamoDB, Amazon Redshift, Amazon QuickSight, Amazon Athena and Amazon Kinesis, and leverage best practices to design big data environments for security and cost-effectiveness.

This course is recommended for:

  • Data Scientists
  • Architects working on Data Science Projects
Security Engineering

The Security Engineering course demonstrates how to efficiently use AWS security services to stay secure in the AWS Cloud. The course focuses on the security practices that AWS recommends for enhancing the security of your data and systems in the cloud. It highlights the security features of AWS key services including compute, storage, networking, and database services. You will also learn how to leverage AWS services and tools for automation, continuous monitoring and logging, and responding to security incidents.

This course is recommended for:

  • Anyone using or developing on AWS

Azure certification levels

The Home Office Public Cloud Team works closely with AWS to provide certification and exam bookings for any of the AWS certification exams. Please find details below.

AWS Cloud Practitioner

This fundamental-level course is intended for individuals who seek an overall understanding of the AWS Cloud, independent of specific technical roles. It provides a detailed overview of cloud concepts, AWS services, security, architecture, pricing, and support.

The AWS Certified Cloud Practitioner examination is intended for individuals who have the knowledge and skills necessary to effectively demonstrate an overall understanding of the AWS Cloud, independent of specific technical roles addressed by other AWS Certifications.

Format: Multiple Choice
Time: 90 mins
Exam Cost: Free
Recommended length of experience: Six months of fundamental AWS Cloud and industry knowledge or 6 hours of online courses
How to book the exam: Please email: aaron.booth3@homeoffice.gov.uk

AWS Associate Certifications

This is the second level of certifications and it is aimed at professionals who might be new to technical roles or have around one years' worth of experience.

There are currently three different certifications at the associate level:

  • The AWS Certified Solutions Architect - Associate examination is intended for individuals who perform a solutions architect role and have one or more years of hands-on experience designing available, cost-efficient, fault-tolerant, and scalable distributed systems on AWS.
  • The AWS Certified SysOps Administrator – Associate examination is intended for systems administrators in a systems operations role with at least one year of experience in deployment, management, and operations on AWS.
  • The AWS Certified Developer - Associate examination is intended for individuals who perform a development role and have one or more years of hands-on experience developing and maintaining an AWS-based application.

Exam rules

Format: Multiple Choice
Time: 130 mins
Exam Cost: Free
Recommended length of experience: One or more years of hands-on experience
How to book the exam: Please email: aaron.booth3@homeoffice.gov.uk

Cloud Academy courses:

  • Solutions Architect
  • SysOps Administrator
  • Developer
AWS Professional Certifications

This is the third level of certification and it is aimed at current professionals with good experience of AWS.

There are currently two separate certifications at the professional level and both are aimed at different job roles. See below:

  • The AWS Certified Solutions Architect – Professional exam is intended for individuals who perform a solutions architect role with two or more years of hands-on experience managing and operating systems on AWS.
  • The AWS Certified DevOps Engineer – Professional exam is intended for individuals who perform a DevOps engineer role with two or more years of experience provisioning, operating, and managing AWS environments.

Exam rules

Format: Multiple Choice
Time: 180 mins
Exam Cost: Free
Recommended length of experience: Two or more years of hands-on experience.
How to book the exam: Please email: aaron.booth3@homeoffice.gov.uk

Cloud Academy courses:

  • Solutions Architect
  • DevOps Engineer Professional
AWS Specialiality Certifications

This is an advanced level of certification for users that have a high level of specialist technical AWS cloud experience.

There are currently five different certifications within the specialist level:

  • The AWS Certified Advanced Networking - Specialty is intended for individuals who perform complex networking tasks.
  • The AWS Certified Big Data – Specialty certification is intended for individuals who perform complex Big Data analyses with at least two years of experience using AWS technology. A new version of the AWS Certified Big Data – Specialty exam will be available in April 2020 with a new name, AWS Certified Data Analytics – Specialty.
  • The AWS Certified Security – Specialty is intended for individuals who perform a security role with at least two years of hands-on experience securing AWS workloads.
  • The AWS Certified Machine Learning - Specialty certification is intended for individuals who perform a development or data science role. It validates a candidate's ability to design, implement, deploy, and maintain machine learning (ML) solutions for given business problems.
  • The AWS Certified Alexa Skill Builder - Specialty certification is intended for individuals who perform a role as an Alexa skill builder. It validates a candidate’s ability to build, test, and publish Amazon Alexa skills.

Exam rules

Format: Multiple Choice
Time: 170 mins
Exam Cost: Free
Recommended length of experience: Several years of AWS experience with hands on experience in the specialist area
How to book the exam: Please email: aaron.booth3@homeoffice.gov.uk

Cloud Academy courses:

  • Security Specialist
  • Machine Learning
  • Advanced Networking
