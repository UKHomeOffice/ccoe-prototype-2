Azure training

Cloud skills challenge

The Home Office has joined up with Microsoft to organise Cloud Skills Challenges. The Cloud Skills Challenge is a fun, free, interactive way to sharpen and develop your Azure skills with the opportunity to get certified across a range of certifications via the Enterprise Skills Initiative(ESI).

Current challenge

The current challenge is Introduction to Azure & Cloud Cost Management (12 Hours over 4 weeks)– Designed for Business Professionals, Product Managers, Business Analyst’s and other non-technical roles.

Join the challenge and sign up using this survey link and we will get back in touch. Home Office users that sign up to the challenge and complete an Azure certification will be nominated for a “Performance Excellence Reward” with the chance of winning a bonus up to £150.

The free training resources for the cloud skills challenge are available on Microsoft Learn. Please sign up using you work email. Microsoft Learn provides:

  • Free, interactive, step-by-step tutorials
  • Hands-on learning coding environments
  • Knowledge checks to validate learning
