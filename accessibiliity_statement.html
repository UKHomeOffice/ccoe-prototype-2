Accessibility

This is our draft accessibility statement for Beta. We will update this page as we do more accessibility testing.

The Home Office - Cloud Centre of Excellence (HO-CCoE) is maintained by the Public Cloud Team (PCT) in Enterprise services. This page explains how the team works to ensure the HO-CCoE is accessible.

Accessibility statement for the HO-CCoE website

This accessibility statement applies to the HO-CCoE at http://www.ccoe-prototype-2.com.s3-website.eu-west-2.amazonaws.com/. Please note this is a temporary link.

The PCT wants as many people as possible to be able to use this website. For example, that means you should be able to:

  • change colours, contrast levels and fonts
  • zoom in up to 300% without the text spilling off the screen
  • navigate most of the website using just a keyboard
  • navigate most of the website using speech recognition software
  • listen to most of the website using a screen reader (including the most recent versions of JAWS, NVDA and VoiceOver)

The team has also made the website text as simple as possible to understand.

AbilityNet has advice on making your device easier to use if you have a disability.

Feedback and contact information

The PCT is always looking to improve the accessibility of this website. If you find any problems that are not listed on this page or think this website is not meeting accessibility requirements, email the PCT at public.cloud@homeoffice.gov.uk

The PCT will review your request and get back to you in 5 working days.

Enforcement procedure

The Equality and Human Rights Commission (EHRC) is responsible for enforcing the Public Sector Bodies (Websites and Mobile Applications) (No. 2) Accessibility Regulations 2018 (the ‘accessibility regulations’). If you’re not happy with how we respond to your complaint, contact the Equality Advisory and Support Service (EASS).

PCT is committed to making its websites accessible, in accordance with the Public Sector Bodies (Websites and Mobile Applications) (No. 2) Accessibility Regulations 2018.

Compliance status

Non-compliance with the accessibility regulations

The HO-CCoE at http://www.ccoe-prototype-2.com.s3-website.eu-west-2.amazonaws.com/ is partially compliant. Some of the noted accessibility issues that need to be addressed are:

  • users cannot navigate most of the website using speech recognition software
  • users cannot listen to most of the website using a screen reader (including the most recent versions of JAWS, NVDA and VoiceOver)

We are woking to fix these accessibility issues.

How this website has been tested for accessibility

The PCT are planning to test the accessibility of the HO-CCoE. We plan to test the service with users with access needs before we move in to Live. We have taken accessibility into consideration from the start and used manual testing and automated testing such as Wave.

How the PCT makes this website accessible

The PCT works hard to ensure that HO-CCoE is accessible.

Where possible, the team aims to research components and patterns with a representative range of users, including those with disabilities.

We also test components to ensure they work with a broad range of browsers, devices and assistive technologies - including screen magnifiers, screen readers and speech recognition tools.

When we publish new content, we’ll continue to make sure that it meets accessibility standards.

Preparation of this accessibility statement

This draft statement was prepared on 09 April 2021.

If you have any questions or need help, contact the PCT by email at public.cloud@homeoffice.gov.uk
